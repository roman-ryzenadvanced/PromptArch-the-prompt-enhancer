## üöÄ Quick Start

1. **Clone & Install**:
   ```bash
   git clone https://github.com/roman-ryzenadvanced/PromptArch-the-prompt-enhancer.git
   cd PromptArch
   npm install
   ```

2. **Configuration**:
   Copy `.env.example` to `.env` and add your API keys:
   ```bash
   cp .env.example .env
   ```

3. **Launch**:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) to begin.
